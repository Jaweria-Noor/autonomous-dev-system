## Requirements Document: Interactive Todo Application

### Functional Requirements

*   **FR1: Todo Checklist Page:** The application shall include a dedicated page displaying a Todo checklist.
*   **FR2: Task Input Field:** The application shall provide an input field (ID: `#todoInput`) where users can type new tasks.
*   **FR3: Add Task Button:** The application shall provide a button (ID: `#addBtn`) that users can click to add a task.
*   **FR4: Task Appending:** Upon clicking the 'Add Task' button, the text from the input field shall be appended as a new item to the Todo list (ID: `#todoList`).
*   **FR5: Input Field Clearing:** After a task is successfully added to the list, the input field (ID: `#todoInput`) shall be automatically cleared.

### Non-Functional Requirements

*   **NFR1: Interactivity:** The application shall provide an interactive user experience for adding Todo items.
